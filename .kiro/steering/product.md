# Product

Triad LMS is a Learning Management System for software engineering education. It provides:

- **Problem Practice**: LeetCode-style coding problem explorer with difficulty filtering, topic tags, and a built-in IDE
- **Labs**: Guided lab exercises with step-by-step modules
- **Learn**: Course catalog and personal course tracking
- **Assessments**: Timed assessments with scoring
- **Leaderboard**: Competitive ranking across cohorts
- **Discussion**: Community discussion board
- **Profile**: User profile and progress tracking
- **Dashboard**: Activity feed, stats (problems solved, streaks, points), and progress charts

Target users are students and instructors. Roles: `student`, `instructor`, `admin`.

The app is currently in a UI-heavy prototype phase — most data is mocked. The real backend is at `http://localhost:5000`.
